# Mermaid Comparison

A simple webapp comparing [beautiful-mermaid](https://github.com/lukilabs/beautiful-mermaid) and [mermaid.js](https://github.com/mermaid-js/mermaid) rendering of the same diagram.

## Demo

Both libraries render an AWS HealthOmics genomic pipeline diagram side by side with theme switching support.

### beautiful-mermaid
- Clean, minimal SVG output
- Easy theming via CSS variables
- 15 curated themes (Tokyo Night, Nord, Dracula, etc.)
- Lighter weight with custom layout engine

### mermaid.js
- Full-featured original library
- More diagram types supported
- Basic built-in themes (dark/default)

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Tech Stack

- Vite
- beautiful-mermaid
- mermaid.js
